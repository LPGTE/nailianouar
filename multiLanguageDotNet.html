Le Multilangues en .Net

1) Introduction

Le Multilangue dans une application informatique désigne la prise en charge des langues multiples pour une application particulière. Toutefois,chaque language de programmation offre une méthode spécifique pour metre en place cette fonctionalité.

Dans le framework .Net, le développement des application multilingue se passe par des fichiers de ressources et la manipulation de l’objet “CultureInfo”.

Dans ce qui suit une présentation Abrégée sur la mise en place de la fonctionalité “Multilanguage” dans les différents technologies .Net.

4) Webforms

  • Créer une page .aspx,
  • Afficher le dossier “App_LocalResources” à l’application à travers le menu “Add / Add ASP Net Folder / App_LocalResources”,
  • Créer un fichier de ressource de la page en cours pour chauque langue,ce fichier de ressource peut être crée manuellement ou bien avec Visual Studio à travers le menu “Tools”/ Generate Local Ressource”,
  • Ajouter le suffixe de la langue au nom de la page, exemple si le nom de la page est “Default.aspx” et qu’on veut créer un fichier de langue pour la langue Anglais, le fichier de ressource aura le nom “Default.aspx.en.resx “ et on aura à la fin deux fichiers de ressources: Un fichier nommé “Default.aspx.resx” pour la langue par défaut et un fichier “Default.aspx.en.resx” pour la langue Anglais,
