StaksNotes

Présentation

Staks Notes est une application de prise de Notes qui peut être utiliser dans un contexte professionnel, académique ou autre. Elle est basé initialiement sur le contrôle Liveswitch et elle est développé pour sauvegarder les notes des utilisateurs sous format des documents manipulables via l’éditeur de l’application.

Elle propose également une méthode d’accès rapide aux notes déjà enregistrés via la méthode de recheche rapide qu’elle dispose

Les données sauvegardés dans l’application peuvent être exportés sous format d’un fichier d’archive zip à travers la fonctionalité d’export comme elle permet aussi de restaurer les donnée sauvegardés à travers la fonction inverse d’import.
